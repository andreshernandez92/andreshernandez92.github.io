# 🏭 SmartBiz Mini ERP

### Zero-Cost Business Automation System in Google Sheets + Apps Script

This repository contains the backend code and architecture documentation for a serverless Enterprise Resource Planning (ERP) system built entirely inside Google Workspace.

It features:
- A distributed multi-spreadsheet architecture
- A real-time POS terminal (HTML/JS Sidebar)
- Automated email and PDF receipt generation
- Role-Based Access Control (RBAC) via Google Sheet Protection API
- Time-driven triggers for daily KPI reporting
