⚙️ How the Automation Works

This workflow acts as a complete lead management system. When a prospect fills out a contact form on any website, the n8n webhook catches the data and runs the entire pipeline automatically.

1

Webhook Trigger

A unique n8n webhook URL catches incoming form data from any website or landing page.

2

Data Validation

A Function node validates the email format and cleans the data before processing.

3

Google Sheets Log

The validated lead is logged as a new row in a Google Sheet with a timestamp.

4

Auto-Response

Gmail sends a personalized welcome email to the lead within seconds.

5

Team Alert

A second Gmail notification is sent to the business owner with the lead's details.

📋 Client Prerequisites for Setup

To deploy this n8n automation for a client, the following setup is required:

n8n Instance

n8n running locally via Docker, npm, or deployed on a VPS (DigitalOcean, Hetzner) for production use with a public webhook URL.

Google Account

A Google account with Gmail and Google Sheets access. OAuth2 credentials must be configured in n8n for secure API access.

Website or Landing Page

A website form that can POST data to the n8n webhook URL. Any HTML form, WordPress plugin, or landing page builder works.

Domain & SSL (Production)

For production deployments, a domain name with SSL (HTTPS) is required to receive secure webhook requests and protect client data.
