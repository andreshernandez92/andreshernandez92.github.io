⚙️ Workflow 1: Lead Router Pipeline

This primary workflow demonstrates complex, multi-step business logic with API integration and conditional branching.

1

Webhook

Catches incoming lead data via POST request.

2

JS Validation

JavaScript validates email, extracts domain, categorizes lead.

3

API Enrichment

HTTP Request calls a REST API to enrich the lead with domain data.

4

IF/Else Router

Branches the flow based on lead category (Enterprise vs SMB).

5

Logs enriched lead data with timestamp and category.

6

Smart Response

Sends a customized email based on lead type.

7

Team Alert

Internal notification with full enriched data payload.

JavaScript Data Manipulation (Code Node)

This Code node demonstrates real-world data transformation — the exact skill hiring managers look for:

// Validate, clean, and categorize the incoming lead const items = $input.all(); const enterpriseDomains = [ 'google.com', 'microsoft.com', 'amazon.com', 'apple.com', 'meta.com', 'netflix.com' ]; for (const item of items) { const email = (item.json.email || '').trim().toLowerCase(); // Validate email format if (!email.match(/^[^\s@]+@[^\s@]+\.[^\s@]+$/)) { throw new Error(`Invalid email: ${email}`); } // Extract domain and categorize const domain = email.split('@')[1]; const category = enterpriseDomains.includes(domain) ? 'Enterprise' : 'Small Business'; item.json = { ...item.json, email, domain, category, timestamp: new Date().toISOString(), source: 'Website Form' }; } return items;

🛡️ Workflow 2: Error Handler

Production automations must handle failures gracefully. This dedicated error workflow catches any failure from Workflow 1, logs it, and sends an urgent alert.

Error Handling Architecture

When any node in the main workflow fails (bad API response, invalid data, Gmail rate limit), this workflow fires automatically.

1

Error Trigger

Fires automatically on any workflow failure.

2

Format Error

JavaScript extracts error message, failed node name, and timestamp.

3

Error Log

Logs the failure to a dedicated "Error Log" Google Sheet tab.

4

Urgent Alert

Sends an urgent email: "⚠️ Workflow failure detected."

📋 Client Prerequisites

To deploy this automation for a client, the following is required:

n8n Instance

n8n running locally (npm/Docker) for development, or deployed on a VPS ($5/mo) for production with a public webhook URL.

Google OAuth2 Credentials

A Google Cloud project with Gmail API and Sheets API enabled. OAuth2 client credentials configured in n8n.

API Key (Free Tier)

A free API key from api-ninjas.com (10,000 requests/month) for the domain enrichment HTTP Request node.

Website Form

Any website or landing page form configured to POST JSON data to the n8n webhook endpoint.

SSL & Domain (Production)

A domain with HTTPS (Let's Encrypt) for secure webhook endpoints in production environments.

Documentation Access

Client provides a list of their software stack, API credentials, and business rules for lead categorization logic.
