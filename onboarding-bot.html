🛠️ Project Overview & Architecture

Agencies and freelancers often lose hours to manual administrative work when signing a new client. This project proves how I utilize **Business Automation** to eliminate data entry and human error.

The architecture consists of a custom HTML/JS frontend interacting with a **Zapier** serverless backend.

[FRONTEND: Your Browser] HTML Form → Javascript fetch() POST ↓ (JSON Payload via Webhook) [MIDDLEWARE: Zapier Workflow] 1. Catch Webhook Data 2. Route payload conditionally based on Service Type ↓ (API Dispatch) [BACKEND CAPABILITIES] ✅ CLICKUP: Create a "New Client" Task and assign to team ✅ GOOGLE DOCS: Duplicate a template and inject {{CompanyName}} ✅ GMAIL: Attach the generated Doc and email the client
